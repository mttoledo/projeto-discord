# projeto-discord

Projeto de criação de landing page desenvolvido para o Bootcamp Santander 2025 - Front-end.

A ideia inicial era criar um site clone do discord que fosse responsivo.

Utilizei flexbox para criar o layout, e fiz uso de media queries (max-width: 500 | 1030 | above) para garantir que o site se adaptasse conforme previsto no modelo do figma.

Fiquei contente com o resultado alcançado, ainda mais considerando o curto tempo que levei para concluí-lo.

## Translate

Landing page creation project developed for Bootcamp Santander 2025 - Front-end.

My initial concept was to develop a responsive website that cloned Discord's interface. 

I implemented Flexbox for the layout and utilized media queries (max-width: 500px, 1030px, and above) to guarantee the site's adaptability according to the Figma model. 

I'm pleased with the outcome, particularly given the short timeframe for completion."
